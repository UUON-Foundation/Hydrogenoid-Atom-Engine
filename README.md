# Hydrogenoid Atom Engine (UUON-HAE-001)

An interactive tool that shows what an electron's probability cloud actually
looks like around a hydrogen-like atom (one electron, any nuclear charge).
Built with React and Three.js.

This does not change or reinterpret physics. It visualizes standard quantum
mechanics that has existed for about a century. The value here is the
visualization and the engineering behind it, not new science.

## What it actually computes

- The radial part of the wavefunction, using the real math (a Laguerre
  polynomial), not a guessed or hardcoded shape.
- The angular part, using the real math (a Legendre polynomial), same idea.
- A point cloud generated by sampling real 3D space so that the density of
  points matches the actual probability density. No shortcuts that distort
  the shape.
- An optional wireframe showing the outer boundary of the cloud, computed in
  a way that is fast because it takes advantage of a real property of these
  wavefunctions (they separate cleanly into a radial part and an angular
  part), not because it's a rough approximation.

## Two ways to view it, and why both exist

- REAL mode: the familiar lobed shapes (like the classic p and d orbital
  pictures). These are what chemists usually draw.
- COMPLEX mode: the mathematically "pure" quantum states. These are always
  ring shaped, never lobed. Most casual visualizations quietly mix these two
  up. This one keeps them separate and labeled, because they represent
  different physical facts.

## What this does not do (stated plainly, not hidden)

- It caps at n = 8. This is a performance limit in how points are sampled,
  not a physics limit.
- The wireframe only shows the outermost layer. Some orbitals have inner
  layers too, and those are not drawn yet.
- A nuclear charge above 1 represents an ion with one electron (like He+),
  not a real multi-electron atom. Electron-to-electron interaction is not
  modeled.
- The gentle motion and camera drift are visual effects added for feel. They
  do not represent anything physically happening to the electron over time.

## Built with

React and Three.js. One self-contained component file. No external assets.
